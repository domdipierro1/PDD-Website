# Stable compact quote form rebuild

The quote form has been rebuilt back to a compact white-card style.

Main fixes:
- Fixed form tile height on desktop.
- Step content no longer stretches the tile.
- Next / Back / Send actions are anchored inside the bottom of the tile.
- Enabled Next / Send buttons are forced blue.
- Step 2 add-ons are compact so they fit inside the tile.
- Property choices use a compact grid.
- Mobile form uses a contained card height with the action button at the bottom of the tile.
