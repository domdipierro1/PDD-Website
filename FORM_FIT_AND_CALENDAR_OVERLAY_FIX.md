# Form fit and calendar overlay fix

Updated the enquiry form so the customer does not need to scroll inside a step:

- Service selection is now a compact two-column grid so all service options fit.
- Property size is now a compact grid so all size options fit.
- Step buttons remain anchored inside the bottom of the form tile.
- Calendar opens as an overlay covering the form step area instead of pushing below the tile.
- Calendar future navigation limit increased to allow long-range future bookings.
