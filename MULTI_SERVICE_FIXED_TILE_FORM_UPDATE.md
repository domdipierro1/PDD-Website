# Multi-service fixed tile form update

Updated the enquiry form flow:

- Step 2 lets customers select one or multiple services.
- Step 3 is optional add-ons and is skipped unless End of tenancy or Deep cleaning is selected.
- Add-ons are Oven cleaning and Interior windows only.
- The form tile uses a fixed internal layout so options fit inside the card.
- Next / Back / Send buttons remain anchored inside the bottom of the tile.
- Primary Next / Send buttons are forced blue.
